FAQ

Will there be an iOS version?

Yes, planned for roadmap v1.0 after Android stabilization.

How to update the app?

Download latest APK from GitHub and install over previous build.

Is this an official university app?

It is a student project for IMES and is under active development.

Will my account password be stored on server?

No. Credentials are stored only on your device if you enable secure auto-login.

Can I install updates over current version?

Yes, if package signature is the same and new version code is higher.

How to report a bug or suggest feature?

Use GitHub Issues from the footer link and attach screenshots if possible.
